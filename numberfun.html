NUMBER OR NUMERIC FUNCTIONS ARE USED TO PERFORM OPERATIONS ON NUMBERS. THEY ACCEPT NUMERIC VALUES AS INPUT AND RETURN NUMERIC VALUES AS OUTPUT.


ABS(N):
THIS FUNCTION RETURNS ABSOLUTE VALUE OF THE COLUMN OR VALUE PASSED.

mysql> select abs(-53) from dual;
            +----------+
            | abs(-53) |
            +----------+
            |       53 |
            +----------+
            1 row in set (0.09 sec)

CEIL(N):
THIS FUNCTION FINDS THE SMALLEST INTEGER GREATER THAN OR EQUAL TO N. WHERE N CAN BE EITHER COLUMN NAME OR EXPRESSION.

mysql> select ceil(88.9) from dual;
            +------------+
            | ceil(88.9) |
            +------------+
            |         89 |
            +------------+
            1 row in set (0.05 sec)

FLOOR(N):
THIS FUNCTION FINDS THE LARGEST INTEGER LESS THAN OR EQUAL TO THE VALUE N. WHERE N CAN BE EITHER COLUMN NAME OR EXPRESSION.

mysql> select floor(88.8) from dual;
            +-------------+
            | floor(88.8) |
            +-------------+
            |          88 |
            +-------------+
            1 row in set (0.02 sec)

MOD(M,N):
THIS FUNCTION RETURNS THE REMAINDER OF M DIVIDED BY N; OR M IF N=0.

mysql> select mod(20,3) from dual;
            +-----------+
            | mod(20,3) |
            +-----------+
            |         2 |
            +-----------+
            1 row in set (0.04 sec)

POWER(M,N):
THIS FUNCTION RETURNS M RAISED TO THE POWER N. THE SECOND ARGUMENT N MUST BE AN INTEGER.

mysql> select power(4,4) from dual;
            +------------+
            | power(4,4) |
            +------------+
            |        256 |
            +------------+
            1 row in set (0.00 sec)

SQRT(N):
THIS FUNCTION RETURNS THE SQUARE ROOT OF N. WHERE N CAN BE EITHER COLUMN NAME OR EXPRESSION.

mysql> select sqrt(25) from dual;
            +----------+
            | sqrt(25) |
            +----------+
            |        5 |
            +----------+
            1 row in set (0.02 sec)

ROUND(M,[N]):
THIS FUNCTION ROUNDS THE COLUMN, EXPRESSION OR VALUE M TO N DECIMAL PLACES.

mysql> select round(90.8535724,2) from dual;
            +---------------------+
            | round(90.8535724,2) |
            +---------------------+
            |               90.85 |
            +---------------------+
            1 row in set (0.02 sec)
