THE CHECK CONSTRAINT DEFINES A CONDITION THAT EACH ROW MUST SATISFY. IT IS USED TO LIMIT THE VALUE RANGE THAT CAN BE PLACED IN A COLUMN.

mysql> create table student3(rollno int, gender char(1) check (gender in('M','F')),
            -> name varchar(20));
