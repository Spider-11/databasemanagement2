CHARACTER OR TEXT FUNCTIONS ARE USED TO MANIPULATE TEXT STRINGS. THEY ACCEPT STRING OR CHARACTER DATA AS INPUT AND CAN RETURN BOTH CHARACTER AND NUMERIC VALUES.

LOWER(STRING):
THIS FUNCTION CONVERTS MIXED CASE OR UPPERCASE CHARACTER STRINGS TO LOWERCASE.

mysql> select lower(name) from student;
            +-------------+
            | lower(name) |
            +-------------+
            | jahid       |
            | harshit     |
            | rahul       |
            | nitin       |
            +-------------+
            4 rows in set (0.03 sec)

UPPER(STRING):
THIS FUNCTION CONVERTS MIXED CASE OR LOWERCASE CHARACTER STRINGS TO UPPERCASE.

mysql> select upper(name) from student;
            +-------------+
            | upper(name) |
            +-------------+
            | JAHID       |
            | HARSHIT     |
            | RAHUL       |
            | NITIN       |
            +-------------+
            4 rows in set (0.00 sec)

CONCAT(STRING1, STRING2):
THIS FUNCTION JOINS TWO STRING VALUES TOGETHER. IT RETURNS STRING1 CONCATENATED WITH STRING2. THIS FUNCTION IS SIMILAR TO || OPERATOR.

mysql> select concat('mohd' ' jahid') as myname from dual;
            +------------+
            | myname     |
            +------------+
            | mohd jahid |
            +------------+
            1 row in set (0.00 sec)

SUBSTR(STRING, M[N]):
THIS FUNCTION RETURNS A SUBSTRING, N CHARACTERS LONG FROM THE STRING SPECIFIED, STARTING AT POSITION NUMBER M. IF N IS NOT SPECIFIED, THE STRING IS EXTRACTED FROM POSITION M TO END.

mysql> select substr(name,2,4) from student;
            +------------------+
            | substr(name,2,4) |
            +------------------+
            | ahid             |
            | arsh             |
            | ahul             |
            | itin             |
            +------------------+
            4 rows in set (0.07 sec)

LENGTH(STRING):
THIS FUNCTION RETURNS THE LENGTH OF A STRING AS A NUMERIC VALUE. THIS FUNCTION IS USUALLY APPLICABLE ONLY WITH VARCHAR2 AND NOT WITH CHAR TYPE COLUMNS.

mysql> select length(name) from student;
            +--------------+
            | length(name) |
            +--------------+
            |            5 |
            |            7 |
            |            5 |
            |            5 |
            +--------------+
            4 rows in set (0.00 sec)

INSTR(STRING,CHAR):
THIS FUNCTION RETURNS THE POSITION OF FIRST OCCURENCE OF 'CHAR' IN 'STRING'.

mysql> select instr(name,'h') from student;
            +-----------------+
            | instr(name,'h') |
            +-----------------+
            |               3 |
            |               1 |
            |               3 |
            |               0 |
            +-----------------+
            4 rows in set (0.10 sec)

ASCII(STRING):
THIS FUNCTION RETURNS THE DECIMAL REPRESENTATION OF THE FIRST CHARACTER OF STRING IN THE DATABASE CHARACTER SET.

mysql> select ascii(name) from student;
            +-------------+
            | ascii(name) |
            +-------------+
            |         106 |
            |         104 |
            |         114 |
            |         110 |
            +-------------+
            4 rows in set (0.00 sec)
