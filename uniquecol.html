THE UNIQUE CONSTRAINT IDENTIFIES EACH RECORD IN A DATABASE TABLE. IT ENSURES THAT THE VALUES ENTERED INTO A COLUMN IS UNIQUE.

            mysql> create table student(rollno int unique, name varchar(20), age int);
        Query OK, 0 rows affected (0.54 sec)

        mysql> desc student;

        | name   | varchar(20) | YES  |     | NULL    |       |
