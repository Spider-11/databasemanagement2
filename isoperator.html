THE SPECIAL OPERATOR 'IS' IS USED WITH THE KEYWORD 'NULL' TO LOCATE NULL VALUES.

mysql> select * from student where rollno is null;
        +--------+---------+-------+---------+
        | rollno | name    | class | section |
        +--------+---------+-------+---------+
        |   NULL | saksham | NULL  | a       |
        +--------+---------+-------+---------+
        1 row in set (0.06 sec)
